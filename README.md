
# Wearthy

A Weather Dashboard which displays real-time weather information and forecasts.

## Features

- Light/dark mode toggle
- Real Weather Data
- Search Weather of Location
- Animation
- Future Weather Predictions

## Tech Stack

**Client:** React

**API:** Rapid API

**Deployment:** Netlify
